# LeetCode Solutions - FAANG Interview Style

## Organization

Each problem is solved following our FAANG interview approach:
1. Google/Meta style question transformation
2. Clarifying questions
3. Template matching
4. Base template → Modified solution
5. Big O analysis
6. Follow-up questions
7. Multiple approaches when available

## How to Use This Repo

1. **Pick a pattern** you want to practice
2. **Read the base template** in `/templates/`
3. **Solve problems** in order of difficulty
4. **Follow the solution format** - clarifying questions, approach, template modification
5. **Review follow-up questions** after solving

## Solution File Naming Convention

```
[pattern-folder]/[problem-number]_[problem-name].md
```

Example: `01-binary-search/875_koko_eating_bananas.md`
